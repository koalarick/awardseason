# Contributing

Thanks for helping improve Academy Awards Pool.

## Quick Start

```bash
# install deps
cd backend
npm install
cd ../frontend
npm install
```

## Quality Checks

```bash
cd backend
npm run lint
npm run format:check

cd ../frontend
npm run lint
npm run format:check
```

## Commit Hygiene

- Keep changes focused and small.
- Update docs when behavior or scripts change.
- Run lint/format checks before opening a PR.
